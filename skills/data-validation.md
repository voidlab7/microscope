# 数据交叉验证

## 触发条件
网络补全完成后自动运行，或用户说「验证数据」「核实一下」

## 输入
- 文件数据 + 网络数据（必填）

## 输出
- 验证报告（每条数据的置信度评级）
- 冲突清单（文件 vs 网络不一致的数据）

## 验证规则
1. 同一字段有 2+ 独立来源一致 → 高置信度
2. 仅 1 个可靠来源 → 中置信度
3. 来源之间有冲突 → 标 ⚠️，列出各方数据
4. 无法验证 → 标 †
