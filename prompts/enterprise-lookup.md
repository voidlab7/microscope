# MicroScope 企业信息查询提示词

---

## 使用场景

当需要补全文件中企业的公开信息时，使用以下搜索策略。

---

## 搜索策略

### 1. 基础信息查询

```
搜索: "{企业全称}" 注册资本 成立时间 法人
搜索: "{企业全称}" site:qcc.com OR site:tianyancha.com
```

提取字段：
- 成立时间
- 注册资本
- 法定代表人
- 注册地址
- 经营范围

### 2. 融资信息查询

```
搜索: "{企业名称}" 融资 轮次 投资方
搜索: "{企业名称}" site:itjuzi.com OR "完成融资" OR "获得投资"
```

提取字段：
- 融资轮次（天使/A/B/C/Pre-IPO/已上市）
- 融资金额
- 投资方
- 融资时间
- 最新估值/市值（未上市取最新轮次估值，已上市取实时市值）

### 3. 产品和业务查询

```
搜索: "{企业名称}" 主营产品 核心技术 业务范围
搜索: "{企业名称}" 官网
```

提取字段：
- 主营产品
- 核心技术/专利
- 目标市场
- 竞争优势

### 4. 最新动态查询

```
搜索: "{企业名称}" 2026 最新 动态 OR 发布 OR 融资 OR 上市
```

提取字段：
- 最新新闻
- 产品发布
- 人事变动
- 战略调整

---

## 输出格式

每家企业输出一张「企业卡片」：

```markdown
### 🏢 {企业名称}

| 维度 | 信息 | 来源 |
|------|------|------|
| 成立时间 | YYYY 年 | [网络:企查查] |
| 注册资本 | X 万元 | [文件] / [网络] |
| 融资阶段 | X 轮 | [网络:IT桔子] |
| 最新融资 | 时间，金额，领投方 | [网络] |
| 核心产品 | 产品名 | [文件] |
| 主要股东 | 股东列表 | [网络] / † |
| 对标企业 | 国际对标 | [分析] |
| 竞争优势 | 一句话 | [文件+网络] |

**分析师备注**: 一句话判断
```
