# MicroScope 数据标注规范

---

## 标注语法

| 标注 | 语法 | 含义 |
|------|------|------|
| 文件来源 | `[文件]` | 数据直接来自用户提供的文件 |
| 网络来源 | `[网络:来源名](URL)` | 数据来自网络搜索，**必须附带完整 URL** |
| 推测 | `[推测:依据]` | 基于已知信息的合理推断，注明推断依据 |
| 未公开 | `†` | 信息未公开，无法通过公开渠道获取 |
| 数据冲突 | `⚠️` | 文件数据与网络数据不一致 |

---

## ⚠️ 强制规则：网络来源必须有 URL

```
✅ 正确：[网络:36氪](https://36kr.com/p/3189304183316353)
✅ 正确：[网络:新浪财经](https://finance.sina.com.cn/stock/xxx)
❌ 错误：[网络:36氪]  ← 缺少 URL，禁止这样写
❌ 错误：[网络]  ← 缺少来源名和 URL
```

如果搜索时忘记保存 URL，标注为 `[网络:来源名,链接待补]`，后续必须补上。

---

## 标注示例

### 正常标注
```
注册资本：1.2 亿元人民币 [文件]
2026 年 3 月完成 Pre-IPO 轮融资 [网络:36氪](https://36kr.com/p/xxx)
估值约 50 亿元 [推测:基于 Pre-IPO 轮融资额和同行业对比]
```

### 未公开
```
前十大股东及股权比例 †
具体融资金额未披露 †
```

### 数据冲突
```
论文发表年份：文件记载 2013 年，实际为 1997 年 ⚠️
（以 ScienceDirect 原文记录为准）
```

## 置信度等级

| 等级 | 标准 | 标记 |
|------|------|------|
| **高** | 多个独立来源一致 | 不需要额外标记 |
| **中** | 单一可靠来源（官方公告/权威媒体） | `[网络:来源]` |
| **低** | 推测或单一非权威来源 | `[推测:依据]` |
| **未知** | 无法获取 | `†` |

---

## 冲突处理规则

当文件数据与网络数据冲突时：

1. **两边都列出**，标 `⚠️`
2. **注明各自来源**
3. **给出分析师判断**（哪个更可信，为什么）
4. **不替用户做最终决定**
